* 初始化git仓库：`git init`
* 配置用户信息：`git config --global user.name "Your Name"` & `git config --global user.email "Your Email"`
* 查看当前状态（暂存、修改等）：`git status`
* 添加文件到暂存区：`git add <file>`（指定文件）`git add .`（所有修改和新增的文件）
* 提交到本地仓库：`git commit -m "提交说明"`
* 查看提交历史：`git log`


# 常见提交英文

**一、前缀**
1. feat 新功能
2. fix 修复bug
3. docs 文档更新
4. style 代码格式整理
5. refactor 代码重构
6. test 增加或修改测试
7. chore 构建过程或辅助工具的变动

1. Add 添加
2. Remove 移除
