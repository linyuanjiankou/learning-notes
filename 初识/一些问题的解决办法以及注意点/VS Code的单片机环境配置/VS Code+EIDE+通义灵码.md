# 配置EIDE

下载vscode，安装插件，包括C/C++，Chinese，Lingma，EIDE，下载完成后一共有13个插件。
在左侧任务栏点击EIDE -> 操作 -> 安装实用工具，下载安装下图中打勾的内容。

点开插件设置，配置编译器路径：将keil v5和v6的对应路径分别插入对应位置，插入完成如下图：![[Pasted image 20260125172052.png]]

# 配置通义灵码

点击左侧通义灵码图标，点击阿里云中国账号登录，按照提示登陆即可

# 后期开发配置
*Tips：每次使用CubeMX重新生成工程后都要进行一次操作*

点击EIDE插件，操作 -> 导入项目 -> MDK -> ARM，将你的工程文件中的MDK文件夹下的keil文件导入进去，新工程要下载对应芯片支持包，烧录器如需更改点击烧录配置更改，添加新的文件夹时点击C/C++属性 -> 包含目录 进行添加。

构筑器选项可以进行类似keil中的设置操作。
