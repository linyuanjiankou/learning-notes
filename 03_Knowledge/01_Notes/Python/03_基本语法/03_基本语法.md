# 注释
在列举说明Python的基本语法之前，我先提一嘴Python的注释。
在C语言中，我们使用 `“//”` 和 `“/*   */”` 分别表示单行注释和多行注释。而Python的注释规则也是分为单行注释和多行注释，具体规则如下：
1. **单行注释**：最常见的注释方式，以井号 `#` 开头，后面紧跟一个空格，然后加上注释内容。`#` 之后的内容直到行尾都会被忽略。
	- **独立成行**：用于解释紧随其后的代码块或逻辑。
	- **行尾注释**：用于解释当前行的代码。
2. **多行注释 / 文档字符串**：虽然Python并没有专用的多行注释语法，但是通常使用三个双引号 `"""` 或三个单引号 `'''` 来包裹多行文本，实现类似效果。
# 变量
1) **变量定义理解**：
	区别于C语言的静态声明，Python采用动态赋值的方式。也就是说：
	
	- 在C语言中，变量在使用前必须显示声明变量的数据类型，编译器在编译时就会确定变量的类型和所需的内存大小，在后续使用中不能随意更改；
	```C
	int age = 25; // 必须声明为 int 类型 
	float pi = 3.14; // 必须声明为 float 类型 
	// age = "hello"; // 编译错误！不能将字符串赋值给int变量
	```
	
	- 在Python中，变量无需提前声明类型，直接通过赋值进行创建，Python解释器会自动判断类型，同一个变量名可以随时被赋予不同类型的值。
	```python
	age = 25 # Python自动推断为整数(int) 
	pi = 3.14 # Python自动推断为浮点数(float) 
	age = "hello" # 合法！age现在是字符串(str)类型
	```

	当然，在Python中你也可以通过强制转换指定数据类型：
	```python
	x = str(3) # x -> '3'
	y = int(3) # y -> 3
	z = float(3) # z -> 3.0
	```

	既然说到了变量的数据类型，Python中的type函数提供了类型获取功能，举个例子：
	```python
	x = 5
	y = 'LiminalStill'
	print(type(x))
	print(type(y))
	```
