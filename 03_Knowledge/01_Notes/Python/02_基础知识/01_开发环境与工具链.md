# 开发环境配置
## 官方Python + PyCharm
作为一个刚开始学习python的初学者，我对开发环境的需求其实并不高，一个功能强大的IDE，一个最基本的解释器和开发库，能够保证快速启动即可，无需考虑环境隔离等问题。
## Windows系统安装
### 官方Python安装步骤
1. 访问python官网：[Welcome to Python.org](https://www.python.org/)
2. 在Download中下载最新版本的安装包：
	- MSIX：初学者使用，可能不完整支持pip，容器化沙盒安装，无需管理员权限
	- MSI：开发者使用，完整包含pip，直接写入系统目录，手动更新新版本，需要管理员权限
	作为初学者，我打算先使用MSIX版本进行安装，之后再去学习其他安装包的东西。
~~3. 右键下载好的msix包，点击“安装”选项进行安装。~~
3. 在配置Obsidian的插件过程中，由于msix包的沙盒导致的复杂环境无法被识别，所以决定卸载msix包下载的python，遂不考虑两种安装包的区别，直接使用 `.exe` 下载python。
### PyCharm下载安装
1. 访问PyCharm官网：[PyCharm，您需要的唯一 Python IDE](https://www.jetbrains.com/zh-cn/pycharm/)
2. 点击下载`.exe`文件
3. 双击安装PyCharm
### 新建项目
1. 我新建了一个名为“Python_learning”的文件夹作为我的第一个项目，保存我之后所有在学习阶段的实验性探索性程序
2. 在新建项目的时候需要配置解释器和虚拟环境
3. 安装pip，并将pip文件所在位置放入环境变量PATH
### 配置镜像源
1. 在cmd中输入指令`pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple`
2. 返回`Writing to C:\Users\用户名\AppData\Roaming\pip\pip.ini`
3. 将该路径添加进环境变量PATH即可
